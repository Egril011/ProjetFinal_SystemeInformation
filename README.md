# Student Project Manager



Si vous voulez voir la base de données, vous pouvez écouter cette vidéo : https://m.youtube.com/watch?v=7TpljNN0IvA, mais au lieu d’en créer une comme il fait à 4:36, vous devez aller sur “Browse”, ensuite ouvrir le dossier Database, puis sélectionner “app.db” et, après l’avoir sélectionné, cliquer sur “Test Connection”. Tout devrait fonctionner.



Sinon, si vous ne voulez pas écouter la vidéo, voici les étapes :



1. Aller dans Manage NuGet Packages Solution, puis installer Microsoft.Data.Sqlite.Core.

2\. Ensuite, aller dans Extensions et installer SQLite and SQL Server Compact Toolbox.

3\. Après l’avoir installé, vous devriez voir le toolbox apparaître.

4\. Faire un clic droit sur Data Connections, puis cliquer sur Add SQLite Connection...

5\. Aller sur Browse, ouvrir le dossier Database, puis sélectionner “app.db”.

6\. Après l’avoir sélectionné, cliquer sur Test Connection.



Pour accéder à l’application, vous pouvez créer un compte ou vous connecter avec l’adresse courriel test et le mot de passe test. Ensuite, vous êtes libre de créer des projets et des tâches pour ces projets.
